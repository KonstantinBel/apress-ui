Vue-компонент селект с фильтром

Подключение:

import filterableSelect from '@apress/apress-ui/dist/common/filterable-select.js';
import '@apress/apress-ui/dist/common/filterable-select.css';

Использование:

<filterable-select
  :options="options"
  :optionId="optionId"
  v-on:change="handleChange"
></filterable-select>

Параметры:

options - [Object{id: Number, value: String, selected: Bool, disabled: Bool}]
optionId - Number
handleChange - Function(Object) (option item)
